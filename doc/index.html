aTpl一款让你急速渲染的模板插件

使用简便(Simple)

你只需要定义好模板，将数据传给他，剩下的事就交给他了。你也可以根据自己的喜好来定义标示符。

效率更高(effect)

模板插件无非使用循环和正则表达式，aTpl根据正则匹配让你的数据在最短的时间内呈献给用户。

无所不能(effect)

aTpl支持多种语法，for循环，条件表达式等等，能够满足您各种业务场景。

文件更小(effect)

aTpl的源码文件大小3.87k，压缩之后的文件只有2.01k。几乎不会影响资源加载时间。

文档

aTpl支持cmd/amd

引用

在你的html页面中添加下面的代码 
<script type="text/javascript" src="aTpl.min.js"></script>
或者使用requireJs 
define(["aTpl"], function () {
    // your code   
});
或者使用seaJs 
define(function () {
    seaJs.use('aTpl');   
});

方法

config
设置表达式开始和结尾标示符 
aTpl.config({
	open: "{{",
	end: "}}"
})
template
加载模板 
aTpl.template("");
render
渲染模板，两个参数。第一个参数为渲染数据，可以是对象也可以是返回对象的函数。第二个参数是渲染完html片段之后执行的函数。 
aTpl.render(data, function() {});
